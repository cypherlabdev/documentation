Send real requests to the AGG API directly from the docs. Select an endpoint below, enter your Bearer token, fill in parameters, and hit **Send**.

<Warning>
Use **staging** credentials when testing. The playground sends real requests to the server defined in the OpenAPI spec.
</Warning>

## How to Use

1. Get an access token by calling `POST /v1/auth/token` with your HMAC-signed credentials â€” see [Authentication](/authentication#obtaining-a-token)
2. Copy the `access_token` from the response
3. Select an endpoint below and paste the token into the **Authorization** field
4. Fill in any required parameters and hit **Send**
