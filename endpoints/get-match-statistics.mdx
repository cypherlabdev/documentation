---
title: "Get Match Statistics"
description: "Retrieve match statistics and weather data"
api: "GET /matches/{id}/statistics"
playground: "simple"
---

Retrieve the possession, shots, corners, fouls, and structured weather data for a match.

<Info>
This is a dedicated endpoint for statistics. It returns exactly the same `statistics` and `weather` data as `GET /matches/{id}?include=statistics,weather`, but without the core match fields.
</Info>

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  Unique match ID (e.g. `prov-60154495`)
</ParamField>

### Example Request

```bash
curl https://api.agg.example.com/v1/matches/prov-60154495/statistics \
  -H "Authorization: Bearer <access_token>"
```

---

## Response

<ResponseField name="code" type="integer">
  HTTP status code (`200`)
</ResponseField>

<ResponseField name="message" type="string">
  `"success"`
</ResponseField>

<ResponseField name="data" type="object">
  Statistics and weather data

  <Expandable title="Statistics Object">
    <ResponseField name="statistics" type="object">
      Match statistics. All values are objects containing `{ "home": int, "away": int }`.
      <Expandable title="Stats Breakdown">
        <ResponseField name="possession" type="object">Ball possession % (home + away = 100)</ResponseField>
        <ResponseField name="shots_on_target" type="object">Shots on target</ResponseField>
        <ResponseField name="shots_off_target" type="object">Shots off target</ResponseField>
        <ResponseField name="corners" type="object">Corner kicks</ResponseField>
        <ResponseField name="yellow_cards" type="object">Yellow cards</ResponseField>
        <ResponseField name="red_cards" type="object">Red cards</ResponseField>
        <ResponseField name="dangerous_attacks" type="object">Dangerous attacks</ResponseField>
        <ResponseField name="fouls" type="object">Fouls committed</ResponseField>
        <ResponseField name="offsides" type="object">Offsides</ResponseField>
        <ResponseField name="attacks" type="object">Total attacks</ResponseField>
        <ResponseField name="free_kicks" type="object">Free kicks</ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="weather" type="object">
      Match weather conditions.
      <Expandable title="Weather Object">
        <ResponseField name="condition" type="string">Weather condition</ResponseField>
        <ResponseField name="temperature" type="integer">Temperature (Â°C)</ResponseField>
        <ResponseField name="humidity" type="integer">Humidity (%)</ResponseField>
        <ResponseField name="wind_speed" type="integer">Wind speed (km/h)</ResponseField>
        <ResponseField name="wind_direction" type="string">Wind direction</ResponseField>
        <ResponseField name="pressure" type="integer">Pressure (hPa)</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
