---
title: "Get Match Insights"
description: "Retrieve H2H, form guide, rankings, and injury reports"
api: "GET /matches/{id}/insights"
playground: "simple"
---

Retrieve head-to-head match history, recent form, league rankings, and team injuries.

<Info>
This is a dedicated endpoint for insights. It returns exactly the same `insights` data as `GET /matches/{id}?include=insights`, but without the core match fields.
</Info>

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  Unique match ID (e.g. `prov-60154495`)
</ParamField>

### Example Request

```bash
curl https://api.agg.example.com/v1/matches/prov-60154495/insights \
  -H "Authorization: Bearer <access_token>"
```

---

## Response

<ResponseField name="code" type="integer">
  HTTP status code (`200`)
</ResponseField>

<ResponseField name="message" type="string">
  `"success"`
</ResponseField>

<ResponseField name="data" type="object">
  Match insights data

  <Expandable title="Insights Object">
    <ResponseField name="home_rank" type="integer">
      Home team league rank
    </ResponseField>
    <ResponseField name="away_rank" type="integer">
      Away team league rank
    </ResponseField>
    <ResponseField name="home_form" type="array">
      Last 5 results (W/D/L) for the home team
    </ResponseField>
    <ResponseField name="away_form" type="array">
      Last 5 results (W/D/L) for the away team
    </ResponseField>
    <ResponseField name="h2h" type="array">
      Last 5 head-to-head matches between the two teams
      <Expandable title="H2H Match Object">
        <ResponseField name="date" type="string">Match date (YYYY-MM-DD)</ResponseField>
        <ResponseField name="home_team" type="string">Home team name</ResponseField>
        <ResponseField name="away_team" type="string">Away team name</ResponseField>
        <ResponseField name="home_score" type="integer">Home team score</ResponseField>
        <ResponseField name="away_score" type="integer">Away team score</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="home_injuries" type="array">
      Home team injuries
      <Expandable title="Injury Object">
        <ResponseField name="player_name" type="string">Player name</ResponseField>
        <ResponseField name="type" type="string">Injury type</ResponseField>
        <ResponseField name="return_date" type="string">Expected return date</ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="away_injuries" type="array">
      Away team injuries (same format as home)
    </ResponseField>
  </Expandable>
</ResponseField>
