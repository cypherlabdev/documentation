---
title: "Get Match Weather"
description: "Retrieve weather conditions at the match venue"
api: "GET /matches/{id}/weather"
playground: "simple"
---

Retrieve the weather conditions at the venue for a specific match, including temperature, humidity, wind, and atmospheric pressure.

<Info>
This is a dedicated endpoint for weather. It returns exactly the same `weather` data as `GET /matches/{id}?include=weather`, but without the core match fields.
</Info>

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  Unique match ID (e.g. `prov-60154495`)
</ParamField>

### Example Request

```bash
curl https://api.agg.example.com/v1/matches/prov-60154495/weather \
  -H "Authorization: Bearer <access_token>"
```

---
