---
title: "List Matches"
description: "Retrieve a paginated list of matches with filtering and sorting"
api: "GET /matches"
playground: "simple"
---

Returns a paginated list of matches. Supports filtering by sport, league, status, and date, as well as sorting.

## Request

```http
GET /v1/matches
Authorization: Bearer <access_token>
```

### Query Parameters

<ParamField query="sport_id" type="string">
  Filter by sport ID
</ParamField>

<ParamField query="league_id" type="string">
  Filter by league ID
</ParamField>

<ParamField query="status" type="string">
  Filter by match status. One of: `live`, `scheduled`, `completed`, `halftime`, `postponed`, `cancelled`
</ParamField>

<ParamField query="date" type="string">
  Filter by date in `YYYY-MM-DD` format
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Items per page (max `100`)
</ParamField>

### Example Request

```bash
# Live football matches, page 1
curl "https://api.agg.example.com/v1/matches?sport_id=1&status=live&page=1&limit=20" \
  -H "Authorization: Bearer <access_token>"

# All matches on a specific date
curl "https://api.agg.example.com/v1/matches?date=2024-03-15" \
  -H "Authorization: Bearer <access_token>"
```
